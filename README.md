#  Web Scraping - Coleta de Dados do X (Twitter)

Este projeto foi desenvolvido como uma atividade acadêmica para coletar posts do X (Twitter) utilizando **Python** e **Selenium**.

##  Objetivo
Extrair de forma automatizada os 4 posts mais recentes de um perfil específico, coletando:
- Autor
- Texto do Post (Descrição)
- Data da publicação
- Salvamento em formato `.csv`

## Tecnologias Utilizadas
- [Python 3.x](https://www.python.org/)
- [Selenium](https://www.selenium.dev/) - Automação de navegador
- [Pandas](https://pandas.pydata.org/) - Estruturação de dados
- [WebDriver Manager](https://pypi.org/project/webdriver-manager/) - Gestão automática de drivers
